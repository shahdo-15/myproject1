# 🛠️ مشروع نظام تسجيل الدخول باستخدام Laravel

## 📄 وصف المشروع
هذا المشروع هو **نظام تسجيل دخول كامل** باستخدام **Laravel**، ويتيح للمستخدمين:
- تسجيل مستخدم جديد (`register`)
- تسجيل الدخول (`login`)
- عرض صفحة ترحيب بعد تسجيل الدخول (`welcome`) تحتوي على الاسم،  والعنوان




## ✨ المميزات
| الميزة | الوصف |

| تسجيل مستخدم جديد | التحقق من صحة البيانات وحفظها في قاعدة البيانات |
| تسجيل الدخول | التحقق من البريد وكلمة المرور باستخدام Hash |
| حماية الصفحات | منع الوصول لصفحة الترحيب بدون تسجيل الدخول |
| عرض البيانات | عرض الاسم، البريد الإلكتروني، والعنوان في صفحة الترحيب |
| تسجيل الخروج | إعادة توجيه المستخدم لصفحة تسجيل الدخول |
| تصميم مرن | صفحات نظيفة وبسيطة بدون مكتبات خارجية |



## ⚙️ متطلبات المشروع
| المتطلب | الوصف |

| PHP | >= 8.1 |
| Composer | إدارة الاعتماديات |
| MySQL | قاعدة البيانات |
| Laravel | >= 10 |
| سيرفر محلي | XAMPP أو أي سيرفر يدعم PHP وMySQL |



## 📂 هيكل المشروع
| المسار | الوصف |

| `app/Http/Controllers/UserController.php` | يتحكم في تسجيل الدخول والتسجيل وعرض الترحيب |
| `app/Http/Middleware/AuthCheck.php` | حماية الصفحات من الوصول بدون تسجيل الدخول |
| `database/migrations/` | يحتوي على ملفات إنشاء الجداول (users) |
| `resources/views/register.blade.php` | صفحة تسجيل مستخدم جديد |
| `resources/views/login.blade.php` | صفحة تسجيل الدخول |
| `resources/views/welcome.blade.php` | صفحة الترحيب بعد تسجيل الدخول |
| `routes/web.php` | تعريف المسارات (Routes) |




## 🖥️ صفحات المشروع
| الصفحة | الحقول | الوصف |
| **Register** | الاسم، البريد، كلمة المرور، العنوان | صفحة إنشاء حساب جديد مع التحقق من الحقول |
| **Login** | البريد، كلمة المرور | صفحة تسجيل الدخول مع رسائل خطأ عند البيانات الخاطئة |
| **Welcome** | الاسم، العنوان | صفحة الترحيب بعد تسجيل الدخول مع زر تسجيل الخروج |



## 🚀 خطوات تشغيل المشروع محليًا
| الخطوة | الأمر |

| استنساخ المشروع | `git clone https://github.com/shahdo-15/myproject1.git` <br> `cd myproject1` |
| تثبيت الاعتماديات | `composer install` |
| إنشاء نسخة من ملف البيئة | `copy .env.example .env` |
| توليد مفتاح التطبيق | `php artisan key:generate` |
| إعداد قاعدة البيانات | فتح `.env` وتعديل `DB_DATABASE`, `DB_USERNAME`, `DB_PASSWORD` <br> `php artisan migrate` |
| تشغيل السيرفر المحلي | `php artisan serve` <br> الوصول الى register
